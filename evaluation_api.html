Módulo de Avaliação de Modelo¶

Esta seção detalha os componentes responsáveis pela avaliação do modelo.

Avaliação¶

Módulo de avaliação de modelo utilizando uma abordagem Orientada a Objetos.

A classe ModelEvaluator encapsula todo o processo de avaliação, incluindo o carregamento de artefatos, geração de predições e cálculo de métricas de desempenho.

Classes de Avaliação: - ModelEvaluator: Resgata os dados e avalia os resultados em teste.

class fundamentos_engenharia_software.evaluation.evaluation.ModelEvaluator(model_path, x_train_path, x_test_path, y_train_path, y_test_path, top_features)¶

Bases: object

Encapsula o processo de avaliação do modelo de machine learning.

Parameters:

  • model_path (str) – Caminho para o arquivo do modelo (.joblib).

  • x_train_path (str) – Caminho para o CSV de features de treino.

  • x_test_path (str) – Caminho para o CSV de features de teste.

  • y_train_path (str) – Caminho para o CSV de alvos de treino.

  • y_test_path (str) – Caminho para o CSV de alvos de teste.

  • top_features (list) – Lista com os nomes das features mais importantes.

Variables:

  • model (Any) – O modelo de machine learning carregado.

  • X_test (pandas.DataFrame) – DataFrame com os dados de teste.

  • y_test (pandas.Series) – Series com os alvos de teste.

  • metrics (dict[str, float]) – Dicionário com os resultados da avaliação (AUC, KS, etc.).

evaluate()¶

Orquestra o processo completo de avaliação do modelo.

Return type:

Dict[str, float]

Returns:

Um dicionário contendo as métricas calculadas.
